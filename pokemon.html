Pokémon Generator

Description

This Pokémon Generator was my capstone project during my Web Development 3 course. It was designed to test and integrate the skills I acquired from Web Development I, II, and III into a real-world application.


Using HTML, CSS, and JavaScript, I created a responsive and dynamic Pokémon Generator. The project utilized the PokeApi to display a gallery of Pokémon with images. Users could load more Pokémon, view details, and mark them as 'caught'. Clicking on a Pokémon revealed a larger image along with its information. Below 'Catch 'button allowed users to add Pokémon to their caught list, which could be managed by scrolling down to view and remove caught Pokémon. This project made me use my web development skills, showing everything I learned in the program at the time.
