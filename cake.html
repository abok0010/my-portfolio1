Cake Website Mockup Build

Description

I designed and developed this website for a midterm assignment during my Design Foundation course in college. The task was to design a site using our current skills, and in the next semester, during the Web Development course, we brought that design to life using only HTML and CSS. The site was also required to be mobile-responsive.


This project fueled my passion for web development. It was my first fully functional website, and I was pleased with how closely it matched the original mockup design. Throughout the process, I learned the effective use of CSS grid and flex, valuable skills for designing websites.
